# Bifract Documentation

Bifract is an open source log management, detection, and collaboration platform built for security teams.

![Bifract search view](images/homepage.png)

It stores logs in ClickHouse for fast search across billions of events, uses PostgreSQL for configuration and collaboration features, and provides a pipe-based query language called Quandrix.

## Where to Start

- **New to Bifract?** Start with [Installation](getting-started/installation.md)
- **Sending logs?** See [Ingestion](ingestion/overview.md)
- **Writing queries?** Learn [Quandrix basics](quandrix/basics.md)
- **Setting up alerts?** Read [Alerting](alerting/alerts.md)
- **API integration?** Jump to the [API Reference](api.md)
